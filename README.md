# Budget PM – INTZA Project

CLI εφαρμογή για ανάλυση κρατικού προϋπολογισμού.

## Εκτέλεση
mvn clean package
mvn exec:java

## Βασικές εντολές
- show summary
- show categories
- set year <YEAR>
- set value <CAT> <VALUE>
- increase / reduce
- compare years <Y1> <Y2>
- scenario <NAME> <PERCENT>
- export csv <YEAR>
- save / load
